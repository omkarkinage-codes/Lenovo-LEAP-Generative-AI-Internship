# EduEco AI - Multi-SDG Conversational Agent (AI Capstone Project)

## 📌 Project Overview
This repository contains the full operational deployment files for my **AI Capstone Project**, completed as the primary milestone of my virtual internship in **Generative AI & Agentic Systems Engineering**. This program is hosted under the **Lenovo LEAP NextGen Scholar Program** (an initiative of Lenovo, implemented by BharatCares, in association with AICTE).

This multi-tenant conversational prototype serves as the core implementation phase of the internship, demonstrating practical technical expertise in conversational state management, node routing architectures, and prompt boundary conditions before proceeding to the final Skill India certification assessment.

The application spans across these core pillars:
* 🌿 **SDG 13 (Climate Action):** Features an interactive branching logical decision matrix that collects user transit modes and dietary selections to formulate 6 distinct, highly customized personal carbon footprint calculation scores.
* 📚 **SDG 4 (Quality Education):** Provides immediate access to high-yield cognitive learning and focus strategies, including structured Pomodoro intervals, Active Recall parameters, and Spaced Repetition curves.
* 💧 **SDG 6 (Clean Water and Sanitation):** Injects actionable domestic resource conservation benchmarks, helping users track and audit daily water efficiency protocols.

## 🛠️ System Architecture & Workflow
Instead of linear form-filling, this system acts as a multi-tenant agent. The conversation flow processes as follows:
1.  **Central Router:** A landing menu greets the user and segments intentions using a robust, clean button selection array.
2.  **Context-Aware Branches:** Safely guides the user through individual SDG question nodes and outputs clear, structured response strings on the viewport.
3.  **Unified Looping:** Every terminal result node gracefully loops back to the master router node to eliminate unhandled conversation breaks or dead ends.
